## Usuários Windows
Podem copiar os códigos para projetos novos em Visual Studio, DevC++, CLion
CodeBlocks e compilar diretamente na IDE.

Caso queiram compilar manualmente, abra um *cmd*:
```BASH
cd [pasta_do_projeto]
g++ [nome arquivo] -o [nome_programa].exe
[nome_programa].exe
```
